***     RetoSophos      ***
Este proyecto es una automatizacion de la pagina Linio.pe, el cual consiste en la solución
 al realizar una busqueda de un producto y agregarlo al carrito; verificando que el nombre
 y los precios sean correspondientes.

El proyecto esta construido en lenguaje Java de tipo Gradle, usando framework Screenplay con Serenity;
por lo cual, se hace uso del framework Screenplay para la estructuracion y codificacion, cumpliendo
los principios SOLID, sin embargo las evidencias seran generadas por Serenity.


PRE-REQUISITOS:
 - Se requiere Internet para descargar las librerias usadas con Gradle y la ejecucion de la automatizacion.
